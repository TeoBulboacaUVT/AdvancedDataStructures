# Advanced Data Structures - Laboratory Assignments

This repository contains laboratory assignments for the **Advanced Data Structures** course. Each lab includes implementation files for various data structure concepts and comprehensive test suites to verify the correctness of the implementations.

## Repository Structure

Each lab directory contains:
- **Implementation files** - Core data structure implementations (usually `.cpp` or language-specific source files)
- **Test files** - Comprehensive test suites that validate all implemented features

## How to Run the Labs

### Prerequisites
- C++ compiler (g++, clang, or MSVC)
- Standard C++ library

### Running Test Files

To run any lab's test file:

1. Navigate to the lab directory:
   ```bash
   cd Lab<X>
   ```

2. Compile the test file with the implementation:
   ```bash
   g++ -o test testfile.cpp <implementationfiles.cpp>[...]
   ```
   (Replace filenames with the specific lab's test and implementation files)

3. Execute the compiled test file:
   ```bash
   ./test
   ```
   Or on Windows:
   ```bash
   test.exe
   ```
